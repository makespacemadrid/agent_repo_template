# Gaps y deuda técnica

> Tier T3: crear cuando se difiere el primer item. Cada entrada debe ser accionable.

## Formato

Cada item debe incluir:
- **status**: `open` | `in_progress` | `resolved` | `deferred`
- **owner**: quién lo atiende
- **source**: dónde se detectó (`ruta/archivo:línea` o contexto)
- **next_action**: qué hacer para resolverlo
